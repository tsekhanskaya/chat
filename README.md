# Chat

### Description

Simple chat with Hotwire, Turbo and Stimulus.

* Ruby 3.0.3
* Rails 7.0.4
* PostgreSQL 15.1

### Launching

1. Download or clone repo. Install all dependencies.

```bash
$ bundle install
```

2. Prepare database

```bash
$ bin/setup
```

3. Start server

```bash
$ bin/dev
```
